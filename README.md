# Sneaker Talk 👟

A full-stack social media platform built for sneaker enthusiasts to connect, share opinions, discuss releases, and engage with the sneaker community.

## Overview

Sneaker Talk was developed to create a dedicated online space where sneaker enthusiasts can interact beyond traditional retail and news websites. Users can create accounts, build profiles, participate in discussions, and engage with a growing sneaker-focused community.

## Features

* User Authentication and Registration
* User Profiles
* Create, Edit, and Delete Posts
* Community Discussions
* Responsive User Interface
* Secure Database Integration
* File/Image Upload Support
* Role-Based User Management

## Tech Stack

```text

```


## Installation

Clone the repository:

```bash
git clone https://github.com/chambersjoshua1999/Sneakertalk2026.git
```

Install dependencies:

```bash
npm install
```

Start the backend:

```bash
cd api
npm start
```

Start the frontend:

```bash
cd front
npm start
```
